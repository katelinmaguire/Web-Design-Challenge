Latitude vs. X


This dataset and these visualizations show weather information from the Open Weather Map API. We used the API to collect data from 546 cities to examine changes in Maximum Tempreature, Cloudiness, Humidity, and Wind Speed on 01/05/2017. We created these four plots in MatPlotLib to examine trends in these four variables as latitude changes.

Click on one of the visualiztions on the right to learn more about these trends. You can also navigate to these pages by clicking the Plots dropdown menu above.

For a comparison of all four plots, visit the Comparison page and to view the full dataset, visit the Data page.
